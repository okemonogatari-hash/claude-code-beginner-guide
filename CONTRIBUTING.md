# Contributing

このリポジトリは、リベシティ「AI初心者サポートチャット」運営有志で育てる Claude Code 初心者向け虎の巻です。

編集前に、まず [`README.md`](README.md)、[`PROJECT_RULES.md`](PROJECT_RULES.md)、[`SOURCES.md`](SOURCES.md) を読んでください。

## 編集の流れ

1. 最新の `main` を確認する
2. 変更対象のファイルだけを編集する
3. 参照元リンクを追加した場合は `SOURCES.md` も更新する
4. コミットメッセージに何を変えたかを書く
5. GitHub Pages の公開URLで表示を確認する

小さな誤字修正やリンク修正はそのままコミットOKです。章構成、ファイル名、公開URL、デザイン方針を変える場合は、事前にチャットで相談してください。
