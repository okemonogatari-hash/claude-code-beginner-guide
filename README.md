## ✍️ 編集の仕方（Git操作不要）

### A. ブラウザだけで編集する（おすすめ）

1. このリポジトリのページを開く → https://github.com/okemonogatari-hash/claude-code-beginner-guide
2. キーボードで **`.` キー** を押す → Web上で VS Code相当の編集画面が立ち上がる
3. `index.html` を開いて編集
4. 左サイドバーの「Source Control」アイコン → コミットメッセージ入力 → 「Commit & Push」
5. これで保存完了。GitHub Pagesも自動で再公開される（1〜2分）

### B. GitHub.com 上で直接編集する

1. `index.html` のページを開く
2. 右上のえんぴつアイコン（Edit this file）をクリック
3. 編集 → 下部の「Commit changes」で保存
